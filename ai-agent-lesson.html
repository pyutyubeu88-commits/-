DAY 1 · 6월 27일

AI 기초 + 나만의 Custom GPT 만들기

코드 없이, ChatGPT를 내 비서처럼 커스터마이징하는 날

🧠 AI가 뭔지 — 딱 5분만
핵심 한 줄: AI는 "엄청 똑똑한 자동완성"이다.
인터넷의 모든 글을 읽고, 다음에 올 말을 예측하는 기계.

❌ 흔한 오해

"AI가 생각을 한다"
"AI가 인터넷을 본다"
"AI는 항상 정확하다"

✅ 실제 작동 방식

패턴 학습 → 예측 출력
학습 데이터 기준 (컷오프 있음)
틀릴 수 있음 — 검증 필요

그래서 우리가 할 일: AI한테 "어떻게 대답해라"를 미리 알려주면 → 내 상황에 맞는 맞춤 비서가 된다. 이게 오늘 만들 Custom GPT의 핵심.
⚙️ ChatGPT 기본 세팅 & 사용법
  • 1

    chat.openai.com 접속 → 로그인

    무료 계정은 GPT-3.5, 유료(Plus $20/월)는 GPT-4o 사용 가능

    💡 무료로도 Custom GPT 만들기 가능. 오늘 실습엔 충분해요.
  • 2

    기본 대화 연습 — 좋은 질문 vs 나쁜 질문

    AI 답변 품질은 질문 방식에 달려있음

    ❌ 나쁜 질문

    "마케팅 잘하는 법 알려줘"

    ✅ 좋은 질문

    "인스타그램 운영하는 강남 카페 사장님을 위한 이번 주 게시글 아이디어 3가지 줘"

  • 3

    프롬프트 3요소 기억하기

    역할(Role) + 상황(Context) + 요청(Task)

    역할: "너는 SNS 마케터야"
    상황: "내 고객은 30~40대 여성, 강남 피부과"
    요청: "인스타 캡션 3개 작성해줘"
🤖 Custom GPT 만들기 — 핵심 실습

Custom GPT = 내가 원하는 성격·역할·지식을 미리 입력해둔 나만의 AI 비서.
한 번 만들면 매번 긴 설명 없이 바로 쓸 수 있음.

  • 1

    좌측 메뉴 → "Explore GPTs" → "+ Create" 클릭

    ChatGPT Plus 필요. 무료 계정은 "My GPTs" 메뉴에서 접근

  • 2

    "Configure" 탭 선택

    Name, Description, Instructions 입력

  • 3

    Instructions에 아래 템플릿 붙여넣기

📝 Custom GPT Instructions 템플릿 — "일정 & 메모 비서"
너는 나의 개인 비서야. 이름은 "아리"고, 항상 친근하고 간결하게 답해. [역할] - 일정 정리 및 요약 - 할 일 목록 작성 - 메모 구조화 - 중요한 내용 핵심만 정리 [말투] - 반말 금지, 존댓말 사용 - 답변은 최대한 짧고 명확하게 - 리스트 형식 선호 [내 정보] - 이름: [본인 이름 입력] - 직업: [본인 직업 입력] - 주요 관심사: [관심사 입력] [행동 규칙] 1. 답변 시작 전 "✅ 정리했어요!" 붙이기 2. 애매한 내용은 바로 질문하기 3. 중요한 날짜/숫자는 굵게 강조
  • 4

    오른쪽 Preview에서 테스트

    "이번 주 할 일: 병원 예약, 친구 생일, 보고서 제출"이라고 입력해보기

  • 5

    "Save" → "Only me" 선택 후 저장

    이제 좌측 메뉴에서 언제든 이 비서 불러올 수 있음

    🎉 첫 번째 AI 비서 완성!
✍️ 프롬프트 핵심 공식

🔑 공식 1 — 역할 부여

"너는 [전문가]야. [상황]에서 [대상]을 위해 [결과물]을 만들어줘."

🔑 공식 2 — 제약 조건

"~하지 마, ~글자 이내로, ~형식으로"
제약이 있어야 원하는 답이 나옴

🔑 공식 3 — 예시 제공

"이런 느낌으로: [예시]"
예시 1개가 설명 100줄보다 낫다

🔑 공식 4 — 출력 형식 지정

"표로 정리해, 번호 리스트로, 한 줄 요약 먼저"
형식을 정하면 쓰기 편해짐

DAY 2 · 6월 28일

AI 에이전트 + Make.com으로 자동화 비서 완성

AI가 스스로 판단하고 행동하게 만드는 날

🧩 AI 에이전트란 뭔가?
에이전트 = AI + 행동력
단순 답변(ChatGPT) → 실제로 뭔가를 하는(Agent) 것의 차이

📱 일반 ChatGPT

질문 → 대답
내가 직접 복붙해야 함
매번 수동 실행

⚡ AI 에이전트

이벤트 발생 → AI가 판단 → 자동 실행
내가 없어도 돌아감
여러 앱 연결 가능

실생활 예시:

이벤트 발생
고객이 인스타 DM을 보냄
AI 판단
메시지 내용 분석 → 문의 유형 분류
자동 실행
맞춤 답장 초안 작성 → 내 폰에 알림
⚙️ Make.com — 노코드 자동화 핵심 툴

Make.com = 앱과 앱을 연결하는 "파이프라인" 도구.
드래그&드롭으로 자동화 흐름 만들기. 월 1,000번 실행 무료.

  • 1

    make.com 접속 → 로그인 → "Create a new scenario"

    시나리오 = 하나의 자동화 흐름

  • 2

    첫 번째 모듈(트리거) 선택

    오늘 실습: Google Sheets → 새 행 추가될 때 시작

    트리거 = "언제 시작할건가"를 정하는 것
  • 3

    + 버튼 → OpenAI(ChatGPT) 모듈 추가

    시트의 내용을 AI에게 넘겨서 처리하게 함

  • 4

    AI 결과물을 다시 시트에 저장

    Google Sheets "Update a Row" 모듈 추가

    입력 → AI 처리 → 출력. 이 3단계가 자동화의 전부.
🔨 실습: AI 비서 에이전트 완성

오늘 만들 것: "할 일 입력 → AI가 우선순위 정리 → 시트에 자동 저장"

📋 Make.com OpenAI 모듈 System Prompt
너는 일정 관리 전문 비서야. 입력된 할 일 목록을 받아서 아래 형식으로 정리해줘: 출력 형식: 1. [긴급+중요] 할 일 이름 — 이유 한 줄 2. [중요] 할 일 이름 — 이유 한 줄 3. [나중에] 할 일 이름 — 이유 한 줄 규칙: - 반드시 이 3가지 카테고리로만 분류 - 각 항목은 한 줄씩 - 설명 없이 바로 리스트 출력 - 한국어로 답변
📊 Google Sheets 구조
A열 B열 C열
입력 날짜 할 일 목록 (내가 입력) AI 정리 결과 (자동 채워짐)

  • 1

    Google Sheets 새 파일 만들기

    위 표 구조로 헤더 행 만들기 (A1: 날짜, B1: 할 일, C1: AI정리)

  • 2

    Make.com 시나리오 완성

    Google Sheets → OpenAI → Google Sheets 3단계 연결

    OpenAI API Key 필요. make.com의 Connections에서 설정. (OpenAI 계정에서 발급 가능)
  • 3

    "Scheduling" 켜기 → 매 15분마다 실행

    이제 시트에 내용만 입력하면 AI가 자동으로 정리해줌

  • 4

    테스트 — B열에 오늘 할 일 입력 후 "Run once" 클릭

    C열에 AI 정리 결과 자동 입력 확인

📧
이메일 답장 초안
아래 이메일에 정중하고 간결하게 답장을 써줘. - 3~5문장 이내 - 첫 줄에 감사 인사 - 마지막에 다음 단계 제안 [이메일 내용을 여기에 붙여넣기]
📊
회의 메모 요약
아래 회의 내용을 정리해줘. 출력 형식: **핵심 결정 사항** (3가지 이내) **나의 액션 아이템** (담당자+기한 포함) **다음 미팅 안건** [회의 메모 붙여넣기]
📱
SNS 게시글 작성
아래 정보로 인스타그램 게시글을 작성해줘. 업종: [업종 입력] 내용: [전달할 내용 입력] 분위기: 친근함/전문성/긴급함 중 선택 조건: - 첫 줄은 눈길 끄는 문장 - 이모지 3~5개 - 해시태그 5개 포함 - 저장 유도 CTA로 마무리
📝
긴 글 3줄 요약
아래 글을 3줄로 요약해줘. - 각 줄은 핵심 포인트 하나 - 숫자/데이터 있으면 포함 - 쉬운 말로 [긴 글 붙여넣기]
