인증 후 노출
현재 우규 역삼점 구글 프로필은 기본 정보 등록까지 완료된 뒤 "인증(소유권 확인) 대기" 상태입니다. 지도에 정상 노출되려면 인증을 반드시 완료해야 하며, 사진은 인증과 무관하게 나중에 추가할 수 있습니다. 따라서 ① 인증을 먼저 끝내 노출을 살리고, ② 사진은 촬영 계획을 잡아 순차 보강하는 두 갈래로 진행합니다.
1 인증(소유권 확인) 대기 풀기 지금 / 최우선
인증이 끝나야 지도 노출·정보 수정·리뷰 응답이 모두 열립니다. 사진보다 먼저 처리.
1
비즈니스 프로필 관리 화면 접속
구글 검색창에 "내 비즈니스" 검색 또는 business.google.com 접속 → 등록 시 사용한 구글 계정으로 로그인 → 우규 역삼점 선택
2
"인증" / "지금 인증하기" 버튼 확인
프로필 상단 또는 알림에 인증 요청이 떠 있음. 클릭하면 가능한 인증 방법이 표시됨
3
제공되는 인증 방법 중 가장 빠른 것 선택
계정·업종·등록 상태에 따라 방법이 다르게 표시됨 (아래 비교표 참고)
4
인증 코드 입력 → 완료
전화/문자/영상/엽서로 받은 코드를 입력하면 인증 완료. 보통 입력 즉시 또는 수 분 내 노출 전환
전화 / 문자 인증가장 빠름
  • 대표 번호로 코드 수신 (수 분 내)
  • 당일 인증·노출 가능
  • 매장 전화 수신 가능 시 우선 선택
영상 / 화상 인증
  • 매장 외관·내부·간판 촬영 영상 제출
  • 사진 없어도 즉석 촬영으로 가능
  • 전화 인증이 안 뜰 때 차선
우편(엽서) 인증
  • 매장 주소로 코드 엽서 발송
  • 수령까지 5~14일 소요 (느림)
  • 다른 방법이 없을 때만
주의 — 인증 방법은 구글이 계정·업종에 따라 다르게 제시합니다. 화면에 전화 인증이 보이면 무조건 우선 선택하세요(당일 처리). 엽서만 뜨면 시간이 걸리므로, 발송 신청 후 본 가이드 섹션 4의 사진 촬영을 그동안 병행하면 시간을 벌 수 있습니다.
2 인증 완료 직후 즉시 점검 (사진 없이 가능) 인증 후 바로
사진이 없어도 텍스트 정보부터 완성하면 검색 노출 품질이 올라갑니다.
No점검 항목우규 역삼점 기준확인
1업체명 정확성이자카야 우규 역삼점 (지점명 포함, 타지점 혼동 X)
2주소 정확성서울 강남구 테헤란로19길 19 (지도 핀 위치 확인)
3대표 카테고리이자카야 / 일식주점 (주 카테고리) + 술집·일식당 보조
4전화번호매장 대표 번호 (네이버 등 타 채널과 동일하게)
5영업시간·휴무일실제 운영 기준 입력 (브레이크타임 있으면 함께)
6업체 설명아래 예시 문장 입력 (역삼·직장인·연어/육회 키워드 반영)
7편의·속성예약·포장·단체석·주차 등 해당 항목 체크
업체 설명 예시 — "역삼역 도보 5분, 테헤란로 직장인을 위한 이자카야 우규 역삼점입니다. 신선한 연어와 육회를 중심으로 퇴근 후 식사 한잔, 회식 자리에 어울리는 메뉴를 준비했습니다. 편안한 분위기에서 한 잔 즐기기 좋은 역삼 이자카야입니다."
역삼 도보 5분 30~40대 직장인 퇴근 후 식사·한잔 연어·육회 중심 회식
3 사진 공백 — 당장 어떻게 대응할까 사진 없는 동안
A
사진은 "필수 조건"이 아니다
사진이 없어도 인증·노출·검색은 정상 작동. 다만 사진 있는 매장 대비 클릭률·신뢰도가 낮으므로 공백 기간을 짧게 가져가는 것이 목표
B
스마트폰 즉석 촬영으로 최소 1~2장부터
전문 촬영을 기다릴 필요 없음. 외관(간판) 1장 + 대표 메뉴 1장만 올려도 "사진 0장" 상태는 벗어남. 화질보다 "있는 것"이 우선
C
고객 업로드 사진 활용 가능
방문 고객·리뷰 사진이 자동으로 프로필에 쌓임. 부적절한 사진은 신고로 정리 가능
D
정식 촬영은 섹션 4 계획대로 순차 보강
임시 사진으로 공백을 메운 뒤, 아래 6종 촬영분으로 단계적 교체·추가
다국어 메뉴와의 연결 — 메뉴판 사진은 본사 다국어 메뉴 작업 결과가 나오면 그때 함께 촬영·등록하면 효율적입니다. (메뉴 사진은 섹션 4의 후순위로 두고, 외관·대표메뉴부터 먼저 확보)
4 사진 촬영 계획 (확보 시 순차 등록) 준비되는 대로
우선순위 순. 위에서부터 확보·등록하면 노출 효과가 가장 큽니다.
1. 외관·간판 최우선
간판이 보이는 정면. 손님이 찾아올 때 기준. 낮/저녁 각 1장
2. 대표 메뉴 최우선
연어·육회 등 시그니처. 위에서·정면 클로즈업, 밝게
3. 내부 전경 2순위
좌석·홀 분위기 한눈에. 조명 켜고 정돈 후 촬영
4. 좌석·테이블 2순위
2인석·단체석 등 자리 종류별. 회식 수요 어필
5. 분위기·디테일 3순위
조명·인테리어·술 진열 등 무드 컷
6. 메뉴판 3순위
다국어 메뉴 확정 후 함께 촬영·등록
촬영 팁 — 스마트폰이면 충분. ① 자연광 또는 매장 조명 최대로 ② 가로 촬영 ③ 음식은 접시가 꽉 차게 ④ 한 종류당 2~3장 찍어 가장 잘 나온 것 선택. 보정 과하지 않게.
