소유권 인증 — 가장 중요한 단계
인증을 완료해야 정보를 수정하고 관리할 수 있습니다
📞
방법 1: 전화 인증 ★ 권장
  • 인증 방법 선택 화면에서 "전화" 선택
  • 대표 전화번호로 자동 음성 통화 또는 문자가 옴
  • 안내 받은 6자리 코드를 입력란에 입력
  • 즉시 인증 완료 — 정보 수정 가능 상태로 전환
전화 인증이 가장 빠릅니다. 대표 전화기를 옆에 두고 진행하세요.
✉️
방법 2: 엽서(우편) 인증
  • 인증 방법 선택 화면에서 "엽서" 선택
  • 입력한 주소 서울 강남구 테헤란로19길 19로 구글이 엽서 발송
  • 수령 후 엽서에 적힌 5자리 코드를 구글에 입력
  • 소요 기간: 5~14 영업일
전화 인증이 불가능한 경우에만 선택. 엽서 분실 시 재발송 요청 필요.
인증 코드를 받았을 때 화면을 닫거나 새로고침하지 마세요. 코드는 30분 이내에 입력해야 합니다.

CHAPTER 08
자주 묻는 질문 (FAQ)
등록 과정에서 자주 막히는 상황과 해결 방법
구글 검색에서 이미 우규 역삼점이 등록되어 있어요. 어떻게 해야 하나요?
"이 업체의 소유자이신가요?" 버튼을 클릭해 소유권 청구 진행. 현재 관리자가 없는 경우 전화 인증으로 즉시 소유권 획득 가능. 관리자가 있을 경우 기존 관리자에게 소유권 이전 요청 필요.
인증이 완료됐는데 구글 지도에 아직 안 나와요.
등록 후 실제 지도 반영까지 최대 7일이 소요될 수 있습니다. 그 사이 사진·설명·영업시간 등 최적화 작업을 진행하세요.
구글 리뷰를 삭제할 수 있나요?
업체 측에서 직접 삭제는 불가능합니다. 욕설·허위 사실·광고성 등 구글 정책 위반인 경우 신고 기능으로 삭제 요청 가능. 부당 리뷰는 "부적절한 리뷰 신고" 클릭 → 사유 선택 → 제출.
구글 비즈니스 프로필 관리를 여러 명이 할 수 있나요?
가능합니다. 관리자 페이지 → 설정 → 사용자 관리 → 이메일 주소 초대. 직원이나 컨설턴트를 "관리자" 또는 "소통 담당자" 권한으로 추가 가능.
임시 휴업이나 특별 영업시간 변경은 어떻게 하나요?
관리자 페이지 → 정보 → 특별 영업시간 → 날짜와 시간 추가. 명절·연말연시 등 특별 기간 전에 미리 업데이트 권장. 임시 휴업은 "임시 휴업으로 표시" 기능 사용.
