## 5. 핵심 가설 (1회차 현장 검증)

### 가설 1 — 콘텐츠는 있는데 노출이 안 된다
블로그·인스타를 운영하지만 "압구정왁싱" 카테고리 키워드에서 상위 노출이 안 되는 구조일 가능성.
→ **확인**: 네이버에서 "압구정왁싱", "압구정로데오왁싱", "신사왁싱" 검색 시 S&H 순위 직접 확인

### 가설 2 — 플레이스 리뷰 자산 부족
블로그·인스타 대비 플레이스 리뷰(방문자/블로그 리뷰 수)가 경쟁사 대비 열세일 가능성.
→ **확인**: 플레이스 리뷰 수 vs 사만다·보들·무무 비교

### 가설 3 — 외국인 수요 전면 미포착
구글 비즈니스 미노출 상태. 압구정로데오 외국인 관광객 급증 상권에서 구글 검색 유입 0 추정.
→ **확인**: Google Maps에서 "waxing apgujeong" 검색 시 S&H 노출 여부

### 가설 4 — 슈가링 전문성이 핵심 무기인데 메시지화 안 됨
당근 리뷰에서 "슈가링 숙련자 드물다"는 평가 = 강력한 차별점.
그러나 채널 노출명·키워드에서 이 강점이 전면에 안 보일 가능성.
→ **확인**: 플레이스 소개문구·인스타 바이오에서 "슈가링 전문" 메시지 유무
