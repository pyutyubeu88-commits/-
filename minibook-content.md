# 소상공인 AI 프롬프트북 — 무료 미니북

### 복붙만 하면 끝나는 사장님 AI 비서, 핵심 프롬프트 10개

> 강남구 소상공인을 현장에서 직접 컨설팅하는 AI 컨설턴트가 검증한 프롬프트입니다.
> **ChatGPT, Gemini, Claude 어디에 붙여넣어도 작동**합니다.
>
> **사용법은 딱 3단계**
> 1. 📋 **복붙용 프롬프트**를 그대로 복사해서 AI에 붙여넣기
> 2. ✍️ **`[ ]` 빈칸**을 내 가게 정보로 바꾸기
> 3. 💬 **출력 예시**처럼 결과가 나오면 → 살짝 다듬어 바로 사용

## 01. 단골 만드는 재방문 문자

**이럴 때 써요** — 한동안 안 온 손님에게 "다시 오고 싶게" 만드는 문자를 보내고 싶을 때

**📋 복붙용 프롬프트**
```
너는 단골 관리를 잘하는 [업종] 사장이야.
[기간] 동안 방문하지 않은 고객에게 보낼 재방문 유도 문자를 써줘.
- 가게 이름: [가게명]
- 혜택: [제공할 혜택]
- 분위기: 부담스럽지 않고 친근하게
- 길이: 60자 이내, 이모지 1~2개
3가지 버전으로 만들어줘.
```

**✍️ 빈칸 채우는 법**
- [업종] → `헬스장` / [기간] → `한 달` / [가게명] → `강남바디짐`
- [제공할 혜택] → `PT 1회 무료 체험권`

**💬 출력 예시**
```
1) 회원님 요즘 바쁘셨죠? 💪 강남바디짐에서 PT 1회 무료로 다시 시작해요!
2) 오랜만이에요 회원님 😊 돌아오시면 PT 1회 무료 체험권 드릴게요!
3) 운동 다시 시작하기 딱 좋은 날! 강남바디짐 PT 무료권 기다릴게요 💪
```

## 02. 인스타그램 게시물 캡션

**이럴 때 써요** — 사진은 찍었는데 무슨 글을 써야 할지 막막할 때

**📋 복붙용 프롬프트**
```
[업종] 인스타그램에 올릴 게시물 캡션을 써줘.
- 주제: [오늘 올릴 내용]
- 톤: [친근하게 / 전문적으로 중 택1]
- 마지막에 행동 유도 문구 1줄 + 해시태그 8개
- 길이: 3~4줄
```

**✍️ 빈칸 채우는 법**
- [업종] → `미용실` / [오늘 올릴 내용] → `봄 신상 펌 시술 전후 사진`
- [톤] → `친근하게`

**💬 출력 예시**
```
봄이 오면 머리부터 바꿔야죠 🌸
자연스러운 컬로 얼굴이 확 살아나는 봄 펌!
전후 사진만 봐도 차이가 느껴지시죠?

지금 예약하면 트리트먼트 서비스까지 ✨
📍프로필 링크로 예약하기

#강남미용실 #봄펌 #펌추천 #헤어스타일 #변신 #얼굴형펌 #단골미용실 #강남헤어
```

## 03. 네이버 플레이스 리뷰 답글

**이럴 때 써요** — 손님 리뷰에 정성껏 답하고 싶은데 매번 비슷한 말만 나올 때

**📋 복붙용 프롬프트**
```
손님이 남긴 아래 리뷰에 대한 사장 답글을 써줘.
- 업종: [업종]
- 리뷰 내용: "[손님이 쓴 리뷰 붙여넣기]"
- 진심이 느껴지게, 재방문을 부르는 한 줄 포함
- 2~3문장, 과하지 않게
```

**✍️ 빈칸 채우는 법**
- [업종] → `식당`
- [리뷰] → `"김치찌개 진짜 맛있어요. 다만 좀 짜네요"`

**💬 출력 예시**
```
맛있게 드셔주셔서 정말 감사합니다 😊
간이 조금 강하셨군요, 다음엔 '덜 짜게' 말씀해 주시면
딱 맞게 끓여드릴게요! 또 뵙기를 기다리겠습니다 🙏
```

## 04. 동네 손님 부르는 전단지 문구

**이럴 때 써요** — 전단지·배너에 넣을 핵심 문구(헤드라인)가 필요할 때

**📋 복붙용 프롬프트**
```
[업종] 전단지에 들어갈 문구를 만들어줘.
- 핵심 강조점: [가장 내세우고 싶은 것]
- 타겟: [주요 손님층]
- 큰 제목(헤드라인) 1개 + 소제목 1개 + 혜택 문구 1개
- 짧고 눈에 확 들어오게
```

**✍️ 빈칸 채우는 법**
- [업종] → `태권도장` / [핵심 강조점] → `초등 저학년 자세·예절 교육`
- [주요 손님층] → `초등학생 학부모`

**💬 출력 예시**
```
[헤드라인] 우리 아이, 몸도 마음도 바르게
[소제목]  강남 OO태권도 — 자세·예절·체력 한 번에
[혜택]    이달 등록 시 도복 무료 + 첫 달 50% 할인
```

## 05. 신메뉴·신상품 소개 글

**이럴 때 써요** — 새 메뉴/상품을 손님이 "먹어보고 싶게/사고 싶게" 소개하고 싶을 때

**📋 복붙용 프롬프트**
```
새로 나온 [상품/메뉴]를 소개하는 홍보 글을 써줘.
- 특징: [재료/장점 2~3가지]
- 가격: [가격]
- 침이 고이게(또는 갖고 싶게) 생생하게 묘사
- SNS·매장 게시용으로 4줄 이내
```

**✍️ 빈칸 채우는 법**
- [상품/메뉴] → `흑임자 크림 라떼` / [특징] → `국산 흑임자, 부드러운 크림, 덜 달게`
- [가격] → `5,500원`

**💬 출력 예시**
```
고소함이 폭발하는 흑임자 크림 라떼 🖤
국산 흑임자를 듬뿍, 부드러운 크림은 덜 달게.
한 모금에 입안 가득 퍼지는 진한 고소함!
딱 지금만 5,500원 — 오늘의 커피로 추천해요 ☕
```

## 06. 컴플레인·환불 응대 메시지

**이럴 때 써요** — 화난 손님에게 어떻게 답해야 할지 난감할 때 (감정 상하지 않게)

**📋 복붙용 프롬프트**
```
불만을 제기한 손님에게 보낼 정중한 응대 메시지를 써줘.
- 업종: [업종]
- 상황: [무슨 일이 있었는지]
- 사과 → 해결책 제시 → 관계 회복 순서로
- 변명처럼 들리지 않게, 진심 담아 3~4문장
```

**✍️ 빈칸 채우는 법**
- [업종] → `네일샵`
- [상황] → `시술 이틀 만에 젤네일이 떨어졌다는 불만`

**💬 출력 예시**
```
불편을 드려 진심으로 죄송합니다.
이틀 만에 떨어진 건 분명 저희 시술 문제가 맞습니다.
편하신 날에 방문해 주시면 무료로 다시 해드리고,
다음 시술 때 쓰실 수 있는 할인권도 함께 드리겠습니다. 🙏
```

## 07. 직원 채용 공고

**이럴 때 써요** — 알바·직원 구하는 공고를 빠르게 잘 쓰고 싶을 때

**📋 복붙용 프롬프트**
```
[업종] [직무] 채용 공고를 써줘.
- 근무 조건: [시간/요일]
- 급여: [급여]
- 우대사항: [있으면]
- 좋은 사람이 지원하고 싶게, 따뜻하지만 명확하게
- 알바몬·당근알바에 바로 올릴 형식으로
```

**✍️ 빈칸 채우는 법**
- [업종] → `카페` / [직무] → `홀·음료 파트타임`
- [근무 조건] → `평일 오전 9시~오후 2시` / [급여] → `시급 11,000원`

**💬 출력 예시**
```
☕ [강남 OO카페] 함께 일할 분을 찾아요!

· 근무: 평일 오전 9시~오후 2시 (주 5일)
· 급여: 시급 11,000원 + 식대 제공
· 하는 일: 음료 제조, 홀 응대 (경험 없어도 친절히 알려드려요)
· 이런 분 환영: 밝게 인사 잘하는 분, 오래 함께할 분

편하게 연락 주세요. 차 한잔하며 면접 봐요 😊
```

## 08. 이벤트·할인 기획안

**이럴 때 써요** — "이번 달 뭐 이벤트 하지?" 아이디어가 필요할 때

**📋 복붙용 프롬프트**
```
[업종]에서 [시기]에 할 수 있는 매출 올리는 이벤트 3가지를 제안해줘.
각 이벤트마다:
- 이벤트 이름
- 내용(어떻게 진행)
- 기대 효과
- 예상 비용 (저비용 위주로)
표로 정리해줘.
```

**✍️ 빈칸 채우는 법**
- [업종] → `헬스장` / [시기] → `신년 1월`

**💬 출력 예시**
```
| 이벤트       | 내용                          | 기대 효과        | 비용 |
|-------------|-------------------------------|-----------------|------|
| 새해 다짐 챌린지 | 30일 출석 시 1개월 무료 연장      | 등록↑·이탈 방지   | 낮음 |
| 친구 추천 2+1  | 친구와 등록 시 둘 다 1개월 추가    | 신규 회원 유입    | 낮음 |
| 인바디 무료 측정 | 신규 상담 시 무료 측정+목표 설정   | 상담→등록 전환↑   | 거의 없음 |
```

## 09. 블로그·상세페이지 글쓰기

**이럴 때 써요** — 네이버 블로그나 상세페이지에 올릴 긴 글이 필요할 때

**📋 복붙용 프롬프트**
```
[업종] 블로그에 올릴 글을 써줘.
- 주제: [글 주제]
- 검색해서 들어올 손님이 궁금해할 내용 위주로
- 소제목 3개로 나눠서, 각 2~3문장
- 마지막에 방문·예약 유도 문구
- 자연스럽고 읽기 쉽게(전문용어 최소화)
```

**✍️ 빈칸 채우는 법**
- [업종] → `치과` / [글 주제] → `스케일링 자주 받아도 되나요?`

**💬 출력 예시 (요약)**
```
■ 스케일링, 얼마나 자주 받아야 할까요?
보통 6개월에 한 번을 권합니다. 치석은 칫솔질로...

■ 자주 받으면 이가 시리지 않나요?
일시적으로 시릴 수 있지만 곧 사라집니다...

■ 스케일링은 건강보험이 됩니다
연 1회 보험 적용으로 부담 없이...

→ 강남 OO치과, 오늘 스케일링 예약하고 입속부터 가볍게 시작하세요 😊
```

## 10. 가격·서비스 안내문

**이럴 때 써요** — 손님이 자주 묻는 가격/서비스를 깔끔하게 정리해 보여주고 싶을 때

**📋 복붙용 프롬프트**
```
[업종]의 가격·서비스 안내문을 손님이 보기 쉽게 정리해줘.
- 항목: [서비스1/가격, 서비스2/가격 ...]
- 추천 항목 1개는 '인기'로 표시
- 깔끔한 표 + 맨 아래 안내 문구 1줄
```

**✍️ 빈칸 채우는 법**
- [업종] → `필라테스`
- [항목] → `1:1 개인레슨 8만원, 2:1 듀엣 5만원, 그룹 3만원`

**💬 출력 예시**
```
| 수업          | 가격(1회) | 비고      |
|--------------|----------|-----------|
| 1:1 개인레슨   | 80,000원  | 자세 교정 집중 |
| 2:1 듀엣 ⭐인기 | 50,000원  | 친구와 함께   |
| 그룹 수업      | 30,000원  | 부담 없이 시작 |

* 첫 방문 시 체험 수업 50% 할인! 편하게 문의 주세요 😊
```
