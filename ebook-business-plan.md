## 0. 한 줄 요약

> **"강남구 소상공인을 AI로 직접 바꾸는 현장 컨설턴트"의 검증된 프롬프트를, 누구나 복붙해서 쓸 수 있게 e북으로 판매한다.**

기존 1:1 컨설팅 사업의 **입구(미끼 상품)** 이자 **자동 수익(패시브 인컴)** 채널로 e북을 운영한다.

## 1. 사업 개요

| 항목 | 내용 |
|------|------|
| 사업명 | (가칭) 소상공인 AI 프롬프트북 |

| 차별점 | 현장 컨설팅에서 **실제로 매출·효율을 낸 검증된 프롬프트** |

### 왜 지금, 왜 나인가
- 이미 **강남구 AI 컨설턴트**로 포지셔닝 완료 (profile.html, program.html)
- 컨설팅 현장에서 쌓인 **실전 프롬프트 자산** 보유 (예: 역삼효태권도장 사례)
- e북 = 컨설팅을 못 받는 사람에게도 노하우를 파는 **확장판**

## 2. 시장과 고객

### 2.1 핵심 통찰
프롬프트 e북은 "정보"가 아니라 **시간 절약 + 즉시 결과**를 판다.
ChatGPT에 무료로 물어볼 수 있는데도 돈을 내는 이유 = "검증된 결과물을 복붙만 하면 되니까."

→ **타겟이 좁고 구체적일수록 잘 팔린다.**

| ❌ 안 팔리는 상품 | ✅ 팔리는 상품 |
|------------------|----------------|
| ChatGPT 프롬프트 1000개 모음 | 태권도·헬스장 원장이 인스타 글 5분 만에 쓰는 프롬프트 30개 |
| 만능 프롬프트 대백과 | 동네 카페 사장님용 단골 만드는 카톡 메시지 프롬프트 |

### 2.2 1차 타겟 페르소나
- **누구**: 강남/서울 동네 자영업 사장님 (40~60대, 태권도장·헬스장·미용실·카페·식당·학원)
- **고통**: 인스타·블로그·전단지 카피 쓸 시간/실력 없음, 광고 대행은 비쌈
- **욕구**: "AI 쓰라는데 뭘 어떻게 쓰는지 모르겠다. 그냥 복붙할 거 줘."

### 2.3 시장 진입 순서
1차: 컨설팅으로 만난 업종(태권도/헬스/요식/미용) → 검증 후
2차: 업종 확장 (학원, 소매, 병의원) → 3차: 일반 마케터/프리랜서

## 3. 상품 라인업 (가격 사다리)

수익은 단품 1개가 아니라 **가격 사다리**로 만든다. e북은 컨설팅으로 가는 입구다.

| 단계 | 상품 | 가격 | 역할 |
|------|------|------|------|
| 0 | **무료 미니북** (프롬프트 10개 PDF) | 0원 | 이메일/카톡 리드 수집 |
| 1 | **메인 e북** (프롬프트 50~80개 + 사용법) | 14,900~29,000원 | 주력 판매 |
| 2 | **노션 템플릿팩** (복붙용 + 분기 업데이트) | 49,000원 | 객단가 상승 |
| 3 | **1:1 AI 컨설팅** (기존 사업) | 무료~고가 | 최종 전환 |

### 3.1 메인 e북 구성안 (예시 목차)
1. 들어가며: AI를 5분 만에 직원처럼 쓰는 법 (프롬프트 기초 3원칙)
2. **매출 확대 — 마케팅 프롬프트**
   - 인스타 피드/릴스 대본, 네이버 블로그 포스팅, 이벤트 문구
3. **고객 관리 — 소통 프롬프트**
   - 단골 카톡 메시지, 리뷰 답글, 컴플레인 응대
4. **매장 운영 — 효율 프롬프트**
   - 메뉴/가격 기획, 직원 공지, 예약/문의 자동 답변
5. **광고·홍보 — 전환 프롬프트**
   - 전단지 카피, 당근 동네홍보, 배달앱 소개글
6. 부록: 업종별 맞춤 변수 채우는 법 + 자주 하는 실수 7가지

각 프롬프트는 **[복붙용 프롬프트] + [채워야 할 빈칸] + [실제 출력 예시]** 3종 세트로 제공 → 이게 무료 ChatGPT와의 결정적 차별점.

## 4. 제작 계획

프롬프트 자산은 이미 컨설팅으로 보유 중 → 정리·검수가 핵심. **2주 내 v1 가능.**

| 주차 | 작업 |
|------|------|
| 1주차 | 프롬프트 카테고리 분류 → 50개 선별 → 각 프롬프트 실제 출력 테스트·검수 |
| 2주차 | 디자인(Canva/Notion) → PDF 변환 → 무료 미니북 10개 별도 추출 → 판매 페이지 연동 |

- **제작 도구**: Canva(디자인) / Notion(템플릿·업데이트본) / PDF 내보내기
- **품질 기준**: 모든 프롬프트는 직접 돌려보고 결과가 쓸만한지 확인 후 수록 (신뢰 = 재구매·후기)

## 5. 판매 채널 전략

| 채널 | 수수료 | 장점 | 단점 | 우선순위 |
|------|--------|------|------|----------|
| **자체 랜딩페이지** + 토스페이먼츠/스마트스토어 | 낮음 | 고객 데이터 확보, 컨설팅 연결 | 직접 유입 필요 | ★ 주력 |
| **크몽 / 탈잉** | 높음(~20%) | 결제·정산 자동, 검색 유입, 초기 신뢰 | 고객 정보 못 가짐 | 보조 |
| **클래스101 / 라우드소싱** | 중 | 트래픽 | 심사·경쟁 | 실험 |
| **인스타 / 블로그 / 유튜브쇼츠** | 0 | 이미 운영 중, 무료 미니북 유입 | 시간 투입 | ★ 유입 엔진 |

### 추천 구조
> 인스타/블로그 콘텐츠 → **무료 미니북**으로 리드 수집(카톡채널·이메일) → 메인 e북 판매 → 후기 확보 → 컨설팅 업셀

이 저장소(aiconsultant-two.vercel.app)에 **e북 판매 페이지**를 추가하면 광고비 없이 바로 시작 가능.

## 6. 가격·수익 시뮬레이션

가정: 메인 e북 19,000원, 자체 채널(수수료 약 3%) 기준

| 시나리오 | 월 판매량 | 월 매출 | 비고 |
|----------|-----------|---------|------|
| 보수적 | 30권 | 약 57만원 | 인스타 유입만 |
| 현실적 | 80권 | 약 152만원 | 미니북 리드 + 후기 누적 |
| 공격적 | 200권 | 약 380만원 | 광고 집행 + 채널 다각화 |

+ **진짜 수익은 e북 자체가 아니라 컨설팅 전환**. e북 100명 구매 → 5~10% 컨설팅 문의 전환이 핵심 KPI.

## 7. 마케팅 퍼널 (콘텐츠 → 판매)

```
[인지]  인스타 릴스/블로그: "사장님이 AI로 5분 만에 인스타 글 쓰는 법"
   ↓
[리드]  무료 미니북 10개 → 카톡채널 추가 / 이메일 입력
   ↓
[판매]  미니북 마지막 장 + 후속 메시지로 메인 e북 안내
   ↓
[신뢰]  구매자 후기·실사용 사례 콘텐츠화 → 다시 [인지]로
   ↓
[업셀]  e북 구매자 대상 1:1 컨설팅 제안 (무료 강남구 사업 연계)
```

### 초기 콘텐츠 30일 플랜 (요지)
- 매주 3회 릴스/포스팅: "업종별 프롬프트 1개 무료 공개" → 더 많은 건 미니북에
- 실제 컨설팅 전후 비교(Before/After) 사례 → 신뢰 자산
- 구매 후기·스크린샷 적극 수집·게시

| 시간 부족 | 무재고·자동결제 구조라 운영 부담 최소, 콘텐츠만 꾸준히 |
