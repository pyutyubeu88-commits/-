🔗 STEP 2 — 영상 불러오기
1
유튜브 영상 URL을 붙여넣고 📥 불러오기 클릭
2
댓글 목록이 자동으로 불러와집니다 (최대 30개)
3
대댓글을 달 댓글을 클릭하면 자동 분석이 시작됩니다
✨ STEP 3 — AI 대댓글 생성
1
AI가 댓글을 자동 분석하고 최적 말투를 선택합니다
2
✨ 대댓글 생성하기 클릭 → 대댓글 3가지 버전 자동 생성
3
마음에 드는 버전 📋 복사 후 유튜브에 붙여넣기
⚡ STEP 4 — 일괄 생성 (시간 절약)
1
영상 로드 후 하단 ⚡ 미답글 일괄 생성 섹션 확인

🚀 일괄 생성 시작으로 미답글 전체를 한 번에 처리
3
결과 테이블에서 행별 복사 또는 ⬇️ CSV 저장
