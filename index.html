🔑 STEP 1 — API 키 설정
1
YouTube API 키: Google Cloud Console → YouTube Data API v3 활성화 → API 키 발급

3

🔗 STEP 2 — 영상 불러오기
1
유튜브 영상 URL을 붙여넣고 📥 불러오기 클릭
2
댓글 목록이 자동으로 불러와집니다 (최대 30개)
3
대댓글을 달 댓글을 클릭하면 자동 분석이 시작됩니다
🤖 STEP 3 — 최적 말투 자동 적용
1
AI가 댓글 유형을 자동 분석하고 최적 말투를 선택합니다
2
✨ 대댓글 생성하기 클릭 → 3가지 버전 자동 생성
3
마음에 드는 버전 📋 복사 후 유튜브에 붙여넣기
⚡ STEP 4 — 일괄 생성 (시간 절약)
1
영상 로드 후 하단 ⚡ 미답글 일괄 생성 섹션 확인
2
🚀 일괄 생성 시작으로 미답글 전체를 한 번에 처리
3
결과 테이블에서 행별 복사 또는 ⬇️ CSV 저장
💡 Tip: Google 계정으로 로그인하면 생성된 대댓글을 YouTube에 직접 게시할 수 있습니다!

Q
API 키가 유출되면 어떻게 되나요?
브라우저 로컬스토리지에만 저장됩니다. 서버에 전송되지 않아 안전합니다.
Q
Gemini API 요금이 발생하나요?
무료 티어로 하루 1,500회 제공됩니다. 일반적인 채널 운영에 충분합니다.
Q
YouTube 직접 게시가 안 될 때는?
Google 로그인 후 YouTube Data API v3가 활성화된 프로젝트의 OAuth를 사용해야 합니다.
