# 로컬 비즈니스 AI 마케팅 자립 컨설팅 — 범용 운영 프레임워크

> **업무 한 문장 정의**  
> 오프라인 매장의 강점을 온라인 성과로 전환하는 현장형 AI 마케팅 자립 컨설팅.

> **적용 업종**: F&B(식당·카페·베이커리) / 뷰티(미용실·네일·피부관리) / 의료(병원·약국·한의원) / 교육(학원·스터디카페) / 피트니스(헬스장·PT·요가·무술도장) / 소매(편의점·전문샵) / 생활서비스(세탁소·수선·인테리어) / 기타 로컬 비즈니스

---

## 1. 업무 정체성 (4대 원칙)

| 원칙 | 설명 |
|------|------|
| **설명보다 실행** | 보고서만 전달하지 않고 현장에서 바로 수정·반영 |
| **대행보다 자립** | 점주가 컨설팅 이후에도 직접 운영 가능한 구조를 남김 |
| **추상보다 현장** | 데이터 + 입구 동선·공간 구조·촬영 포인트·서비스 접점까지 직접 확인 |
| **AI 기반 표준화** | 질문지·체크리스트·분석표·실행자료·문구 템플릿으로 반복 가능한 체계 구축 |

---

## 2. 주요 서비스 범위

| 영역 | 설명 |
|------|------|
| 사전 진단 | 기본 정보·채널 현황·상품/서비스 구조·사진 자산·점주 핵심 고민 수집 |
| 리뷰/상권 분석 | 리뷰 언어·강점·불만 요소·경쟁군·가격대·방문·이용 목적 분석 |
| 현장 실행 | 채널 정보 정비·소개 문구 조정·대표 이미지 방향 정리·현장 촬영 판단 |
| 포지셔닝 설계 | 매장을 한 문장으로 설명하는 선택 이유와 핵심 메시지 확정 |
| 자립 구조 구축 | 점주 직접 운영 가능한 체크리스트·문구·기준 데이터·후속 실행안 정리 |

---

## 3. 전형적인 업무 흐름

```
1단계  사전 질문지 배포
2단계  응답 회수 및 기준 데이터 정리
3단계  리뷰·상권·경쟁 분석
4단계  1회차 현장 컨설팅 설계
5단계  현장 정보 정비 및 포지셔닝 확정
6단계  후속 회차용 실행자료 및 숙제 정리
7단계  (장기) 점주 자립 운영 구조 구축
```

## 10. 컨설팅 철학 (변하지 않는 기준)

1. 좋은 가게가 온라인 때문에 손해 보지 않게 한다
2. 광고비를 쓰기 전에 먼저 정보 정비와 포지셔닝을 바로잡는다
3. 강점을 새로 만드는 것이 아니라 이미 있는 강점을 온라인에서 보이게 만든다
4. 점주의 행동 가능성을 높이는 방향으로 문서와 실행 단위를 설계한다
