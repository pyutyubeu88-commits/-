목표: 의뢰인 상황 파악
💬 오프닝 멘트
"안녕하세요! 오늘 짧은 시간 안에 실질적인 변화를 드리려고 준비 많이 해왔어요. 일단 지금 어떻게 공부하고 계신지 먼저 좀 여쭤봐도 될까요?"
📋 파악할 질문 순서
  • IMAT 언제 보세요? 목표 점수나 지원 학교 있으세요?
  • 15년치 기출 중 어떻게 활용하고 싶으세요? 연도별로 풀어보기? 주제별로 묶기?
  • AP Biology 교재는 지금 어떻게 쓰고 계세요? 처음부터 읽기? 챕터 골라서?
  • 지금 Gemini 쓰실 때 가장 불편한 게 뭔가요? 시간이 너무 걸려서? 결과가 마음에 안 들어서?
  • 교재 PDF는 오늘 가져오셨나요?
⚠ 주의
질문을 한꺼번에 쏟아내지 말고 대화하듯 1개씩. 의뢰인 말 끊지 말기. IMAT은 이미 확인됐으니 시험 날짜와 현재 약점 섹션 파악이 핵심.
TIP — IMAT Biology는 11–38번(28문제)으로 비중이 압도적. 오늘 수업 무게중심을 Biology 솔팅과 문제 생성에 두는 게 맞음. AP Biology PDF가 48MB라 Claude 직접 업로드 안 될 수 있음 → 챕터 텍스트 복붙으로 대체.

강사용 진행 스크립트

파트 1 (플로우 재설계) + 파트 2 (플래시카드)

강사 전용 — 비공개
10 – 30분
NotebookLM 실습 — 플로우 재설계
핵심 파트
💬 도입 멘트
"지금 쓰시는 방식이 나쁜 게 아니라, 단계가 너무 많아서 피곤하신 거예요. 오늘 이걸 하나로 줄여드릴게요. 제 화면 보시겠어요?"
🖥 시연 순서 (강사 화면 기준)
  • notebooklm.google.com 열기 → 새 노트북 만들기 클릭
  • "소스 추가" → IMAT 기출 PDF 1개 업로드 (2024.pdf부터 시작)
  • 여러 연도 PDF를 소스로 추가하면 15년치 통합 분석 가능 — 같이 해보기
  • 오른쪽 패널 → "Study Guide" 클릭 → 생성되는 동안 설명
  • 생성 완료 후 채팅창에 "한글 번역 버전도 만들어줘" 입력 → 결과 보여주기
💬 Study Guide 생성 중 설명
"지금 자동으로 요약이랑 Q&A를 뽑고 있어요. 예전에 하시던 Gemini 4단계가 여기서 다 이뤄지고 있는 거예요. 30초면 됩니다."
🖥 의뢰인에게 직접 해보게 하기
  • 의뢰인 노트북으로 본인 Google 계정으로 NotebookLM 접속
  • 본인 영상 링크 직접 붙여넣기
  • Study Guide 직접 눌러보게
  • "채팅창에 뭐든 질문해보세요" → 자유롭게 써보게 유도
⚠ 자주 생기는 문제
AP Biology PDF(48MB)는 NotebookLM 업로드 한도 초과 가능 → 연도별 IMAT 기출 PDF(3~10MB)로 대체. / NotebookLM 서버 느릴 때는 기다리면서 Claude 프롬프트 파트 먼저 진행.
시간 체크 — 20분 지나도 의뢰인이 아직 접속 중이면 "일단 제 화면으로 계속 볼게요" 하고 넘어갈 것
30 – 55분
플래시카드 자동 생성 실습
ChatGPT / Claude
💬 도입 멘트
"NotebookLM에서 뽑은 요약을 이제 플래시카드로 만들 거예요. 복붙 한 번이면 20개가 나와요."
🖥 실습 순서
  • NotebookLM Study Guide 내용 전체 복사
  • ChatGPT 열기 → 프롬프트 파일에서 "프롬프트 A" 복붙 → 내용 붙여넣기 → 실행
  • 결과 나오면 같이 읽어보면서 "이게 Anki 카드 형식이에요" 설명
  • 시간 남으면 프롬프트 B(CSV 형식)도 바로 시연
💬 중간 설명
"이걸 한글로도 받고 싶으시면 프롬프트에 '한글 버전도 함께' 한 줄 추가하면 돼요. 오늘 드리는 프롬프트 파일에 전부 들어있어요."
TIP — 의뢰인이 Anki를 모를 수 있음. "종이 플래시카드 디지털 버전이에요, 앱에서 넘기면서 공부해요"로 설명. 오늘 Anki 설치까지 할 필요 없음.

강사용 진행 스크립트

파트 3 (문제 추출) + 정리 + 비상 대응

강사 전용 — 비공개
55 – 75분
시험 포맷 문제 추출 실습
Claude / ChatGPT
💬 도입 멘트
"이제 제일 직접적인 거예요. 교재 내용 넣으면 시험 문제가 바로 나와요. 준비하시는 시험 스타일에 딱 맞게 조정도 가능해요."
🖥 실습 순서
  • Claude.ai 열기 → 프롬프트 파일에서 "MCQ 프롬프트" 복붙
  • NotebookLM 요약 내용 or 교재 텍스트 붙여넣기 → 실행
  • 결과 확인 → "이 중에 실제 시험에 나올 것 같은 거 있어요?" 의뢰인에게 물어보기
  • 시간 되면 단답형 프롬프트도 바로 실행
💬 Claude Project 설명 (시간 여유 있을 때)
"이걸 매번 붙여넣기 귀찮으시면 Claude Project에 교재 PDF 한 번 넣어두면 학기 내내 '3장 문제 뽑아줘' 한 줄만 치면 돼요. 같이 만들어볼까요?"
시간 부족 시 — 문제 추출은 의뢰인이 집에서 혼자 해볼 수 있음. 프롬프트 파일 드리면 되니까 75분 내로 못 해도 OK. 대신 프롬프트 파일 꼭 전달할 것.
75 – 90분
정리 & 마무리
숙제 확정
💬 마무리 멘트
"오늘 배운 흐름 정리해드릴게요. NotebookLM에 자료 넣고 → 요약/Q&A 뽑고 → ChatGPT로 플래시카드 만들고 → 문제 추출. 이 루틴 한 번만 익히시면 매 강의마다 30분씩 절약돼요."
📋 마무리 체크
  • 수강생용 핸드아웃(lesson1.html) 공유 or 출력본 전달
  • 프롬프트 파일(lesson1_prompts.txt) 카톡으로 전달
  • 숙제 확정: "이번 주 강의 1개로 직접 해보기"
  • "다음에 또 필요하시면 연락 주세요" — 추가 세션 자연스럽게 열어두기
💬 클로징 멘트
"오늘 한 번만 익혀두시면 이제 혼자서 다 되세요. 막히는 부분 있으시면 언제든 연락 주시고요. 화이팅 하세요!"
비상 상황 대응 가이드
상황 대응
의뢰인이 유튜브 링크 안 가져옴 IMAT 기출 PDF로 바로 진행. "기출로 먼저 해보고 나중에 유튜브 추가하면 돼요"
NotebookLM 접속 안 됨 ChatGPT/Claude로 먼저 플래시카드 파트 진행 후 나중에 NotebookLM 재시도
WiFi 느림 기다리는 동안 핸드아웃 함께 읽기. 강사 핫스팟 켜기 (미리 데이터 확인)
의뢰인이 "이미 알아요"라고 함 심화 버전으로 전환: "그럼 복습 자동화 스케줄 세팅 같이 해볼게요"
