오른쪽 패널(Studio)에서 모두 사용 가능

📚 Study Guide 핵심
  • 소스 내용을 섹션별 요약 자동 생성
  • 각 섹션마다 Q&A 문제 자동 포함
  • 생성 후 복사해서 ChatGPT에 붙여 플래시카드로 변환 가능
  • 생성 시간: 약 30초~1분
오른쪽 패널 → Study Guide → Generate
🎙️ Audio Overview 팟캐스트
  • AI 2인 대화 형식의 영어 팟캐스트 자동 생성
  • 이동 중 듣기 → 다운로드 버튼으로 MP3 저장
  • 2인 고정 (남성+여성 AI 목소리)
  • Biology 기출 넣으면 개념 설명 팟캐스트 생성
오른쪽 패널 → Audio Overview → Generate
❓ FAQ 질문 정리
  • 소스에서 자주 묻는 질문 형태로 자동 정리
  • 개념 정리 카드처럼 활용 가능
  • 각 답변에 출처 표시 → 원문 즉시 확인
  • 시험 전 빠른 복습에 적합
오른쪽 패널 → FAQ → Generate
📋 Briefing Doc 요약문서
  • 소스 전체를 문서 형식으로 요약
  • 시험 직전 훑기용 1페이지 요약 생성 가능
  • 복사 → Word/Notion에 붙여넣어 보관
  • 여러 소스를 통합한 브리핑 문서 제작
오른쪽 패널 → Briefing doc → Generate
