### 업무 정체성 (4대 원칙)
1. **설명보다 실행** — 보고서만 전달하지 않고 현장에서 바로 수정·반영
2. **대행보다 자립** — 점주가 컨설팅 이후에도 직접 운영 가능한 구조를 남김
3. **추상보다 현장** — 리뷰·상권 데이터 + 입구 동선·좌석 구조·촬영 포인트·서비스 접점까지
4. **AI 기반 표준화** — 질문지·체크리스트·분석표·실행자료·문구 템플릿으로 반복 체계화

### 주요 서비스 범위
| 영역 | 내용 |
|------|------|
| 사전 진단 | 기본 정보·계정 상태·메뉴/가격·사진 자산·점주 핵심 고민 수집 |
| 리뷰/상권 분석 | 리뷰 언어·강점·불만·경쟁군·가격대·방문 목적 분석 |
| 현장 실행 | 채널 정보 정비·소개 문구 조정·대표 이미지 방향·현장 촬영 판단 |
| 포지셔닝 설계 | 매장을 한 문장으로 설명하는 선택 이유와 핵심 메시지 확정 |
| 자립 구조 구축 | 점주 직접 운영 가능한 체크리스트·문구·기준 데이터·후속 실행안 |

4. 1회차 현장 컨설팅 설계
5. 현장 정보 정비 및 포지셔닝 확정
6. 후속 회차용 실행자료 및 숙제 정리

### 컨설팅 철학
- 좋은 가게가 온라인 때문에 손해 보지 않게 한다
- 광고비 전에 먼저 정보 정비와 포지셔닝을 바로잡는다
- 강점을 새로 만드는 것이 아니라 이미 있는 강점을 온라인에서 보이게 만든다
- 점주의 행동 가능성을 높이는 방향으로 문서와 실행 단위를 설계한다
