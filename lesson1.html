NotebookLM 실습 순서
1
소스 추가
notebooklm.google.com
  • 새 노트북 생성 → + 소스 추가
  • 유튜브 링크 붙여넣기 (자막 자동 추출)
  • PDF 교재 업로드 (드래그 앤 드롭)
  • 소스 여러 개 동시에 넣어도 통합 처리됨
2
자동 생성 기능 활용
오른쪽 패널
  • 요약 (Summary): 핵심 내용 자동 정리
  • Study Guide: 섹션별 요약 + Q&A 자동 생성 ← 오늘 주요 실습
  • FAQ: 자주 나오는 개념 질문 형태로 정리
  • Timeline: 순서가 있는 내용 자동 정렬
3
채팅으로 심화 질문
한국어 OK
  • 채팅창에 한국어로 질문해도 소스 기반으로 답변
  • "이 챕터에서 가장 중요한 개념 3개만 뽑아줘"
  • "이 내용을 시험 문제 5개로 만들어줘"

파트 2 · 플래시카드 & 문제 추출

30분 – 75분

실습
플래시카드 자동 생성 프롬프트
아래 프롬프트를 ChatGPT 또는 Claude.ai에 붙여넣고 학습 내용을 함께 첨부하세요.
프롬프트 A — 기본 Q&A 플래시카드
아래 내용을 바탕으로 플래시카드를 만들어줘.

형식:
Q: [질문]
A: [답변 — 2~3문장, 핵심 용어 포함]

조건:
- 총 20개
- 생물화학 시험에서 자주 나오는 개념 위주
- 영어 Q&A로 작성, 각 카드 아래에 한글 번역 추가

[여기에 NotebookLM 요약 또는 교재 내용 붙여넣기]
프롬프트 B — Anki CSV 형식 (Anki 앱에 바로 가져오기 가능)
아래 내용으로 Anki 카드를 만들어줘.

출력 형식 (탭 구분, 헤더 없이):
[영어 질문]	[영어 답변]	[한글 힌트]

카드 수: 15개
난이도: 중~상 (시험 출제 가능 수준)

[내용 붙여넣기]
시험 포맷 문제 추출 프롬프트
MCQ (객관식 5지선다)
아래 내용으로 MCQ 10문제를 만들어줘.

형식:
Q. [문제]
A) ...  B) ...
C) ...  D) ...  E) ...
정답: [알파벳]
해설: [1~2문장]

난이도: MCAT 수준
주제: glycolysis / TCA cycle 등

[내용]
단답형 / 빈칸 채우기
아래 내용으로 단답형 문제 10개와
빈칸 채우기 문제 5개를 만들어줘.

단답형: Q + 모범답안 (영어)
빈칸: ___로 표시, 정답 별도

[내용]
개념 비교형
"A와 B를 비교하는
표 형식 문제 5개 만들어줘.
(예: 유산소 vs 무산소 호흡)"

[내용]
Claude Project 설정 (한 번만 하면 계속 사용)
